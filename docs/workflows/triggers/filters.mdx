Triggers can be filtered on many properties. They can be categorised as general filtering and event filtering.

- General filtering such as `IP Address`, `URL Path`, etc.
- Event filtering are properties specific to the trigger event, refer to each trigger for the supported filter properties.

### General Filter Properties

- IP Address - The IP address of the user
- URL Path - The URL path that is being hit

### Filter operator

- Equals
- Not Equals
- Matches

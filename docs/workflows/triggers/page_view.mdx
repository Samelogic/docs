The page view event is configured by a single field representing the URL or path that the user is currently viewing.

#### URL

The URL value will match fully qualified URLs such as `https://samelogic.com/docs/triggers`

#### Paths

The path value will match a path that is not part of the hostname or base URL, such as `/docs/triggers`.

#### Wildcards

The wildcard value will match a URL or path with a wildcard such as `/docs/triggers/*`. It will match any subpaths such as `/docs/triggers/click`.
